【详细介绍】如何hexo+github搭建自己的个人博客

最近想搭建属于自己的博客，看了下网上的教程，发现了很多坑，虽然最后一一解决，但还是浪费了不少时间。希望看完我这篇的文章的同学，可以少走点坑。
不会搭建ssh的同学可以看我上篇博客。

前期准备

1，git（网上有下载的），我以前下过具体步骤应该不难，直接安装即可
2，Node.js 的安装（我下的是9.70）算比较新的，我看很多博客都教的是4.0版本。
下载地址：https://nodejs.org/download/release/v9.7.0/
3，gitHub账户的配置：（需要一个邮箱即可，建议不要校内邮箱，验证码可能会拦截）
注册网址：https://github.com/

命令：
npm -v
node -v
git –version

环境配置与操作步骤

1，hexo的安装

在桌面上鼠标右键，点击git bash here（安装好git应该都有）
输入指令：
npm install -g hexo-cli
结果如下图：

2，hexo的初始化

在你的盘中新建个文件夹，在文件夹内点击鼠标右键，点击git bash here。输入以下命令：
hexo init
初始化化文件夹，文件夹内结果如图：

输入：
hexo install
（如果出现下图，说明你以前安装过了，不用管）

再输入：
npm install hexo-deployer-git
安装插件

3，github设置

打开github网址，登录个人账户。
https://github.com/

点击第一个 new respository

大家按照我的进行勾选即可。在第一个创建respository name 下记住：
必须后面的后缀名必须有.github.io。前面你随便填
创建完后，如下图：

复制自己的ssh网址，就是git@那一行
打开_config.yml

修改以下代码：
deploy:
type: git
repo: git@github.com:waq250/waq530.github.io.git
branch: master
第三行，替换成自己的即可，保存
最后在git bush master 输入
hexo g -d
即可。
就可以出现以下结果。
【waq250是我以前创建的，你们的网址根据你们自己命名会有所区别】

1，git安装
2，github账号注册
这两步我就不多说了，很简单

二，上传文件到github

1，github创立项目

点击new repository。出现以下画面，按我的配置即可。命名可以根据自己喜好命

2，ssh连接

在桌面上右键，再点击git bash here，会出现命令管理器
输入下面指令：
ssh-keygen
【注意：第一次安装，记得输入指令后，停顿时要按回车键才会进行下一步】

根据生成的.ssh文件夹路径，寻找到.ssh文件，打开id-rsa.pub文件
【可以直接搜索id-rsa.pub文件】
